
    If python is not installed, install it the latest python release from here (likely the 64 bit version)
    After installation, download a zip file of this code (best placed locally on your computer, like in My Documents)
    Unzip the zip file
    copy the file location of the unzipped folder, open command prompt, and type 'cd '. Replace the text of 'file location' with the file location you just copied
    In the command prompt, paste the following command: pip install -r requirements.txt
    Open the 'Cary spectrophotometer plotting.py' program, click run, and select the .csv file to analyze
    Alternatively, within command prompt, copy and paste this command to execute the python code: cary spectrophotometer plotting.py
